# ⚡ BlitzPrep

> Hızlı İngilizce, Güçlü Başlangıç

İngiltere'ye gidecek bir Apple Store Müdürü için geliştirilmiş İngilizce hazırlık uygulaması.

## Özellikler

- 📚 Kelime Kartları — ~170 kelime, akıllı tekrar sistemi
- 💬 Senaryolar — 20 diyalog (günlük yaşam + Apple Store)
- ✏️ Cümle Kurucu — ~150 cümle yazma pratiği
- 🗣️ Konuşma Pratiği — 10 senaryo
- 🧠 Gramer — 5 konu, ~57 soru
- 🎯 Quiz — Kelime / Gramer / Karışık

## Kurulum

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
